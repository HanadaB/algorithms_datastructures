# Algorithms and Data Structures (ALDABI)
This repository contains C++ implementations of classical algorithms and data structures developed during the course Algorithms and Data Structures for Bioinformatics (ALDABI).
The exercises cover important techniques used in computational biology, such as string matching, dynamic programming, indexing structures, and formal language models.

## Implemented Algorithms
- Horspool algorithm (exact string matching)
- Suffix array construction
- Needleman–Wunsch sequence alignment
- BLAST neighborhood generation
- Q-gram index for approximate matching
- Aho–Corasick trie for multi-pattern matching
- Pushdown automaton (PDA)

## Repository Structure
Each folder includes source code, test files, Makefiles, and the exercise description.

## Tools
C++ and Make  
Developed and tested on Linux/macOS.

## Purpose
The goal of this repository is to practice core algorithmic concepts used in bioinformatics and sequence analysis.
